🌍 Alerta Global — Versión Beta

⚠️ Esta es una versión BETA v1.0. Algunas funciones pueden no estar disponibles o presentar errores.

Términos y Condiciones de Uso

1. Propósito informativo
Alerta Global es una aplicación de carácter informativo. Las alertas mostradas provienen de fuentes públicas oficiales (USGS, NASA, NOAA, Smithsonian, OpenWeatherMap) y se presentan con fines informativos únicamente.

2. No sustituye a las autoridades
Esta aplicación NO reemplaza los sistemas oficiales de alerta temprana. En caso de emergencia, siga siempre las instrucciones de las autoridades locales (SENAPRED, ONEMI, protección civil de su país).

3. Limitación de responsabilidad
Los desarrolladores de Alerta Global no se hacen responsables por decisiones tomadas basadas en la información de esta aplicación. Los datos pueden contener retrasos o imprecisiones.

4. Privacidad y datos
La aplicación puede solicitar acceso a su ubicación para mostrar alertas cercanas. Esta información no se almacena en servidores externos y solo se usa localmente en su dispositivo.

5. Uso de APIs
La aplicación utiliza APIs públicas gratuitas. La disponibilidad de los datos depende de terceros y puede variar.

6. Versión Beta
Esta es una versión de prueba. Algunas funciones pueden estar incompletas. Agradecemos su paciencia y comentarios para mejorar la aplicación.
